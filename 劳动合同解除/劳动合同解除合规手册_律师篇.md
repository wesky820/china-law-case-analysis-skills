# 劳动合同解除合规 律师参考指南

这份文档是给 AI 看的，也可以给代理劳动争议案件的律师自己看。目标是为律师提供法律适用框架、举证责任分配、裁判规则梳理、地域差异对照和类案检索指引。

本指南已包含《最高人民法院关于审理劳动争议案件适用法律问题的解释（二）》（下称《解释二》）最新规定。具体法条、司法解释、地方指导意见可能更新；案件代理前应核验现行规则。

---

## 1. AI 的角色

你是一个帮助律师处理劳动合同解除案件的参考顾问。

用户是律师，可能是：

- 代理劳动者（原告/申请人），要证明公司构成违法解除
- 代理用人单位（被告/被申请人），要证明解除行为合法
- 正在做法律研究，需要了解某类解除的裁判口径
- 正在准备仲裁/诉讼材料，需要梳理举证要点
- 需要评估案件风险，判断胜诉概率

回答时要做到：

- 先给法律适用结论，再展开分析
- 引用法条、司法解释、指导案例、入库案例
- 明确标注地域差异——北京/上海/深圳/天津/江苏/四川等地裁判口径不同
- 指引举证责任分配和证据链构建
- 提供类案检索关键词
- 不替法官做判决，给裁判趋势分析而非确定性结论

---

## 2. 先理清哪几件事

代理任何劳动解除案件前，先确认：

1. 当事人是谁：劳动者（普通/高管/派遣工/平台从业者）vs 用人单位
2. 合同履行地在哪：直接决定适用的地方裁判口径
3. 解除方式是什么：书面通知（载明的理由是什么）、口头通知、行为表示（停发工资/停缴社保/禁止上班）
4. 解除理由归类：第39条？第40条？第41条？第44条？还是协商一致？
5. 程序是否履行：工会通知？书面送达？调查申辩？
6. 是否有特殊保护情形：第42条六类情形、三期、工会成员等
7. 当事人诉求：要求确认违法解除+2N赔偿金，还是要求继续履行+补发工资
8. 时效是否过期：知道或应当知道权利被侵害之日起1年

## 3. 法律适用框架

### 3.1 解除路径与对应法条

```
第36条 协商一致解除 → N
第39条 过错性解除（6种） → 0补偿
第40条 无过错性解除（3种） → N+1或N+30日
第41条 经济性裁员 → N
第44条 劳动合同终止 → N（部分情形）
第87条 违法解除 → 2N 或 继续履行
```

### 3.2 代理不同当事人的审查重点

**代理劳动者（主张违法解除）**：
1. 解除理由是否法定？（"价值观不符"=不成立）
2. 证据是否充分？（单方内部文件=可排除）
3. 制度是否有效？（民主+公示+签收）
4. 程序是否合规？（工会通知+送达+申辩机会）
5. 处罚是否过度？（比例原则+梯度处分）
6. 是否有特殊保护？（第42条）

**代理用人单位（主张合法解除）**：
1. 解除理由是否有法条依据？
2. 制度是否经得起合法性+合理性双重审查？
3. 证据链是否闭环（制度→行为→定性→程序）？
4. 程序是否存在可补正的瑕疵（如起诉前补正工会通知）？
5. 能否适用"过错分担"条款减少补发工资金额（《解释二》第18条）？

## 4. 违法解除的司法认定体系

### 4.1 二元审查框架

法院对解除行为的审查从两个维度展开：

**实体维度**：是否具备法定解除事由 + 事由是否有证据证明
**程序维度**：解除程序是否合法（工会通知+送达+申辩+调查）

任一方面不满足→违法解除

### 4.2 解除事由的排他性与固定性

**排他性**：劳动法体系下解除事由严格法定，排除约定解除。用人单位通过规章制度自行创设解除事由→无效。案例（2023）沪0118民初21325号。

**固定性**：解除通知载明的理由=裁审审查核心基础。事后不得补充或变更。

**多事由策略**：可同时援引多个法定事由，一个成立即合法。

### 4.3 举证责任

- 《司法解释一》第44条：因用人单位解除决定发生的争议，由**用人单位负举证责任**
- 《劳动争议调解仲裁法》第6条：用人单位掌握管理的证据由用人单位提供

## 5. 赔偿金计算的地域差异

### 5.1 基本公式

**2N = 解除前12个月平均工资 × 工作年限 × 2**

### 5.2 各地差异对照

| 争议项 | 北京 | 上海 | 深圳 | 天津 | 四川 | 浙江 |
|--------|------|------|------|------|------|------|
| 加班费计入基数 | ✅ | ❌ | ✅ | - | ❌ | - |
| 年终奖计入 | ✅ | ✅ | ✅ | - | ❌ | - |
| 病假工资 | 包含 | 包含(主流) | 有争议 | - | - | 剔除 |
| 赔偿金分段计算 | 不分段 | 不分段 | 不分段 | **分段** | - | - |
| 未建工会通知义务 | 须通知上级 | 有争议 | 无需(主流) | 无需 | - | 须通知 |
| 客变认定("组织架构优化") | ❌不认可 | ✅可认可 | - | - | - | - |

**规范依据**：
- 北京：京高法发〔2024〕534号第79条（客变定义）+ 解答（一）第55条（基数）
- 上海：高院《民事法律适用问答（2013年第1期）》（基数）+ 沪高法〔2009〕73号第21条（赔偿金）
- 深圳：《裁判指引》第97条
- 天津：《天津法院劳动争议案件审理指南》（分段计算+未建工会）
- 浙江：浙高法民一〔2014〕7号第11条（病假工资剔除）
- 江苏：《江苏省劳动合同条例》第31条（未建工会须通知）

### 5.3 封顶规则

月工资>社平工资3倍→基数封顶3倍+年限封顶12年。赔偿金不分段（最高法入库案例2023-16-2-186-003，天津例外）。

### 5.4 税务

财税〔2018〕164号：≤当地上年职工平均工资3倍部分免征个税。实务中赔偿金参照执行（上海（2025）沪0104执异312号案）。

## 6. 违纪解雇的司法审查

### 6.1 合法性审查：制度有效性

审查链：内容合法→条款清晰→民主程序→公示告知

三个最常见败诉原因：

1. 无法证明民主程序（案例（2024）内02民终1945号、案例（2025）京02民终10517号、案例（2023）沪民申1969号）
2. 未有效公示告知（案例（2022）京0115民初15016号）
3. 条款模糊不清（"其他严重违纪行为"等兜底条款）

### 6.2 合理性审查：解雇正当性

审查维度：主观过错程度+客观情节方式+实际后果与潜在风险+是否遵循比例原则

**比例原则典型案例**：
- 案例（2017）京01民终4436号：跳过了梯度处分直接开除→违法
- 案例（2024）粤01民终12243号：玩手机未达严重程度+无前置警告→违法

### 6.3 四类违纪行为认定

| 违纪类型 | 关键认定要素 | 重点案例 |
|---------|------------|---------|
| 严重违反规章制度 | 制度有效+行为明确+严重程度+程序合规 | 胜诉：(2024)内民申2242号；败诉：(2024)鄂01民终12126号 |
| 严重失职/徇私舞弊 | 岗位职责+主观过错+因果关系+重大损害量化 | (2024)渝民申4430号（损害不以实际发生为要件）；2025上海十大案例之二（职责边界认定） |
| 双重劳动关系 | 制度有无禁止+是否严重影响+是否拒不改正 | (2023)吉05民终276号（无法证明严重影响→违法） |
| 被追究刑事责任 | 以生效判决为准；酌定不起诉≠被追刑 | (2021)津01民终8151号 |

### 6.4 程序公正

调查→告知→申辩→决定。不给人申辩机会=程序重大瑕疵（案例（2020）沪02民终3355号）。

## 7. 不能胜任解除

### 七步审查逻辑

```
制度基础→考核标准→实施考核→第一次告知不胜任→培训/调岗→二次考核→解除程序
```

任一步缺失或瑕疵→违法解除。

### 关键案例

**(2024)京02民终7986号**：三个败诉点——考核依据不明确、调岗通知未载明原因（关联性不成立）、整体数据不等于个人问题。

**某某公司诉谢某案**：仅凭"主管评定"给C级=缺乏客观标准→两次考核均不能成立→违法。

## 8. 客观情况重大变化

### 北京 vs 上海（核心分歧）

**北京**（极度严格）：客变限于不可抗力、政策变化、特许经营变化三类。案例（2022）京03民终4701号："客变是外部不可控变化，非经营状况不佳或管理模式改变。" 案例（2024）京03民终17274号/17255号、(2023)京0105民初46442号均不支持。

**上海**（相对宽松）：组织架构调整如属整体性、非针对性的企业重大经营决策，可构成客变。案例（2023）沪01民终1980号、(2024)沪01民终873号均支持。

### 协商变更的司法审查

审查是否善意诚信：方案是否具体合理、是否给足考虑时间、是否以维系劳动关系为出发点。

**典型案例：爱某公司诉张某案（上海2023年度优秀裁判文书）**——17分钟协商+2.7万降至1万+无具体岗位=不诚信协商；亏损从103亿缩窄=未发生"根本性变化"。

### 客变证据体系

1. 外部客观背景（审计报告、财务数据+因果关系论证）
2. 纲领性决策文件（董事会/股东会决议，整体性调整）
3. 充分告知记录（全员宣布+一对一沟通+工会说明）

## 9. 工会通知程序

### 法律框架

《劳动合同法》第43条：单方解除必须**事先**通知工会。

补正规则（《司法解释一》第47条）：**起诉前**补正有效，起诉后补正无效。

北京特别规定（2026年5月起）：须**提前5个工作日**通知。

### 关键案例

- (2024)苏01民终5966号：二审中才通知→超期→违法
- (2025)沪02民终6426号：未建工会未通知上级+起诉后补正→违法。**法院明确指出**："未建工会不应成为免除通知义务的理由，否则形成逆向激励。"
- (2019)粤0112民初1442号：广州观点——未建工会无需通知

### 律师策略

未建工会地区→保守策略：主动通知当地街道工会/区总工会。有(2025)沪02民终6426号的判决趋势支持。

## 10. 恢复劳动关系：《解释二》新规则

### 五大变化

| 变化 | 旧规则 | 新规则（第16条） | 支撑案例 |
|------|--------|----------------|---------|
| 岗位被替代 | 可阻断恢复 | 不再构成独立抗辩 | (2026)京02民终3507号 |
| 入职新单位 | 可阻断恢复 | 需"严重影响"或"拒不解除" | (2025)京03民终16099号 |
| 信赖关系破裂 | 可作抗辩 | 主观因素被排除 | (2025)京03民终15143号 |
| 退休年龄 | 达到=不能恢复 | 享受养老保险待遇=不能恢复 | - |
| 事实劳动关系 | 不能恢复 | 可请求恢复 | (2025)豫0482民初10912号 |

### 工资补发与过错分担（第18条）

用人单位按正常劳动工资标准补发。双方都有过错的→各自承担→为用人单位提供了抗辩空间。

## 11. 平台用工法律定性

### 三类关系认定

| 关系类型 | 认定标准 | 解除后果 |
|---------|---------|---------|
| 劳动关系 | 劳社部发〔2005〕12号三要素 | 适用《劳动合同法》全责 |
| 不完全劳动关系 | 八部门意见（人社部发〔2021〕56号） | 不直接适用违法解除赔偿；可能有违约/公平补偿责任 |
| 民事关系 | 《民法典》 | 按合同约定 |

### 支配性劳动管理判断（指导案例239/240号）

**构成劳动关系**：时间控制+内容控制+过程控制+规则控制+收入来源+业务外观。
**不构成**：基于平台维护的必要运营管理，从业者可自主决定接单/时段/平台。

### "去劳动关系化"的司法应对

- 签承揽/合作协议≠规避劳动关系（指导案例237号）
- 注册个体工商户≠规避劳动关系（指导案例238号）
- 连环外包→认定"关系最密切"企业为用人单位（指导案例238号）

### 多主体复合用工的地域差异

| 地区 | 劳务派遣责任 | 外包责任 |
|------|------------|---------|
| 上海 | 派遣单位与平台连带 | 平台过错承担 |
| 天津 | 平台承担用工单位责任 | 合作企业主责+平台过错承担 |
| 山东/重庆 | 平台承担用工单位责任 | 平台承担相应责任（裁量空间大） |
| 四川 | 派遣单位+平台各担其责 | **平台先行承担清偿责任**（最严格） |

## 12. 劳务派遣解除责任分配

- 法定解除权归属派遣单位（用人单位）
- 用工单位仅能退回，不能直接解除
- 违法退回+违法解除→连带赔偿（《劳动合同法》第92条）
- 派遣单位因用工单位退回而解除→协议通常约定实际赔偿由用工单位承担

## 13. 高管解聘双轨制

### 公司法 vs 劳动法

| 轨道 | 适用法 | 核心规则 | 司法审查范围 |
|------|--------|---------|------------|
| 公司法 | 《公司法》第67条 | 董事会无因解聘权 | 仅审查程序合法性，不审查理由真实性（指导案例10号） |
| 劳动法 | 《劳动合同法》 | 严格法定条件+程序 | 全面审查实体+程序 |

### 关键裁判规则

- 指导案例10号：董事会解聘高管属公司自治，程序合法即有效
- (2017)京01民终431号：免职+明确告知劳动关系存续+继续付薪=不构成违法解除

## 14. 电子证据采信规则

### 法院审理思路

证据链完整性+证明力实质性+取证程序规范性+内容连续一致性。

**胜诉 vs 败诉对比**：
- 败诉（2024）沪0115民初3978号：单方邮件+陈旧孤证=无闭环
- 胜诉（2023）沪0115民初101789号：微信+第三方记录+调查录音=完整闭环

### 举证指引

四阶段证据体系：制度依据层→行为固定层→定性对应层→程序履行层。

应对员工常见抗辩："聊天记录被断章取义"（展示完整上下文）、"账号被盗用"（提供IP/设备日志）、"邮件非本人"（展示该邮箱的连续使用记录）。

---

## 15. 送达效力

解除通知以送达为生效要件。未有效送达=解除行为未完成。

送达优先级：直接送达（签收）→邮寄（EMS）→电子送达（须合同事先约定地址）→公告送达（最后手段）。

案例（2019）内民申2380号：未尝试直接/邮寄+径行公告=程序不合法。

---

## 16. 诉讼策略要点

### 代理劳动者
重点攻击：解除理由非法定、制度未经民主程序、证据为单方内部文件、无申辩机会、未通知工会、处罚违反比例原则。

### 代理用人单位
重点防御：制度链完整（民主+公示+签收）、证据链闭环、程序可证、工会通知已补正（起诉前）。

《解释二》新策略：
- 主张不能恢复→举证客观障碍
- 员工入职新单位→发催告函+证明严重影响
- 过错分担→争取减少补发工资

## 18. 类案检索与法条核验

### 法条核验

使用 MCP 工具 `yuandian_law_vector_search`：
- 核实引用的法条是否现行有效、是否有更新
- 检索地方性法规和指导意见
- 通过 `fatiao_filter` 按时效性、效力级别、实施日期过滤

### 案例检索

使用 MCP 工具 `yuandian_case_vector_search`：
- 输入自然语言描述的核心争议点
- 通过 `wenshu_filter` 按案由、法院层级、地域、日期过滤
- **强制限定合同履行地**——同一案由在不同地区的裁判结果可能完全相反
- 关注入库案例、指导案例、典型案例的参照效力

### 核心地域差异提醒

| 问题 | 差异 |
|------|------|
| 客变认定 | 北京❌ vs 上海✅ |
| 赔偿基数(加班费) | 北京深圳✅ vs 上海❌ |
| 赔偿基数(年终奖) | 大部分✅ vs 四川❌ |
| 赔偿金分段 | 天津分段 vs 全国不分段 |
| 未建工会通知 | 北京江苏✅ vs 天津❌ vs 上海有争议 |

> ⚠️ 做法律意见时，必须首先确认劳动合同履行地，然后检索该地区的类案裁判口径。不能以单一地区的裁判结果推及全国。
